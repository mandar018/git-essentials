git commands :

1. git status - to check the current status of repo
2. git clone git@github.com:<your GitHub username>/cs102-git-essentials.git - to clone the repo
3. git add file_name.extension - to add file in staging area
4. git commit -m "type a message to name the changes" - to commit changes made in repo
5. git push origin main - to push the changes to server repo
